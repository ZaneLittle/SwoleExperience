# SwoleExperience

SwoleExperience is an all-in-one Weight Tracker, Macro Tracker and Workout Planner!

### Current State of Things
The Weight Tracker section is currently under development and close to release!

The Macro Tracker and Workout Planner is coming soon, but the development effort has not yet started, so stay tuned for updates!
